Поставка решения включет в себя два каталога:
 - bin
 - sources

Каталог bin содержит исполняемы файлы и настройки приложения:
 - javatask-1.0.jar
 - application-default.yml
 - run-show-db
 - run-write-in-db
Каталог sources содержит исходный код приложения. Стуктура файлов стандартная для maven и gradle.

Для запуска приложения используются скрипты run-show-db и run-write-in-db.
run-show-db - выводит в консоль и лог-файл содержимое базы.
run-write-in-db - записывает раз в секунду текущее значение timestamp

Предварительные условия:
 1. На системе в которой планируется запуск приложения должна быть установлена Java вресии не младше 1.8
 2. Должен быть доступ к базе MySQL
 3. Необходимо отредактировать application-default.yml и указать реквизиты для подклчючения к базе MySQL

При первом запуске приложение самостоятельно создаст схему и таблицы для работы в базе.
